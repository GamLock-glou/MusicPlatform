# Music Platform

## Tech stack

+ Next
+ Nest
+ MongoDB
+ TypeScript

## libraries in frontend

+ redux/toolkit
+ mui
+ sass
+ axios
+ next-redux-wrapper

## libraries in backend

+ schematics
+ mongoose

## How to start?

```node
  yarn install
  yarn start
```
